# Super Bowl Project

## Overview
This is a Python program I created to apply what I've learned in class and outside study. The program lets users to look up Super Bowl winners and MVPs by year or search how many times a team has won.

## What This Program Does
- Finds the Super Bowl winner for a specific year
- Finds the Super Bowl MVP for a specific year
- Shows how many championships a team has won

## Skills Used
- Dictionaries
- Functions
- Loops
- Conditional statements
- Basic error handling
